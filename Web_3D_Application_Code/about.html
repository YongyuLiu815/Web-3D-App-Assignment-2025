About

Users can experience animated on/off effects and interactively switch lighting colors, highlighting the diversity and aesthetic charm of each design.

3D Models

To create this website, I designed four lamp models using Blender. During the modeling process, I utilized UV editing and shading tools to enhance the visual appearance of each lamp. I also applied lighting effects through the shading system, allowing the models to emit a soft glow in dark environments, creating a moody and atmospheric presentation.

Design

The design of this page is built using pure HTML, CSS, and Three.js without relying on external UI frameworks. A custom layout splits the screen into two sections: a 3D rendering area and a control panel. The navigation and interface are fully hand-coded, providing greater flexibility for styling and responsiveness. The dark theme with golden accents aligns with the overall aesthetic of the site. Three.js is used to load and display the 3D lamp model. Lighting is set up using AmbientLight and DirectionalLight. Users can interactively rotate the model, toggle lights, adjust light intensity and color, and enable a wireframe view to examine its structure.

Integration

On the homepage, I integrated preview images for each lamp model and used CSS to control their layout and size. Each image is clickable, allowing users to navigate directly to the corresponding 3D model display page. In addition, the navigation bar also provides quick access to the same models through categorized dropdown links, ensuring a consistent and intuitive user experience.

Interaction

I added interactive OrbitControls to allow users to rotate, zoom, and explore the 3D model freely within the web page. Additionally, I implemented control buttons that let users toggle rotation, adjust lighting, and switch between different model views or modes. These interactions enhance engagement and give users more control over how the lamp is experienced.

Implementation

The project has been successfully completed and deployed, and all the expected features (including model presentation, lighting control, rotation and interaction) work well in modern browsers. During the development process, I encountered some challenges, such as how to adjust the lighting effects imported from Blender, how to maintain compatibility on different devices, and how to optimise the camera and controllers for smoother user operation. Through continuous testing and debugging, the system is now stable and responds well to user interactions. Initial user feedback is positive, especially in terms of visual aesthetics and lighting interaction, which enhances the realism and cultural expression of the model.

Deeper Understanding

Beyond the technical implementation, this project gave me a deeper appreciation for how cultural artifacts can be translated into digital form. By modeling traditional Chinese and European lamps, I reflected on how material, form, and light work together to create mood and identity. Recreating these effects digitally required not only 3D skills, but also sensitivity to cultural context and user experience. I also realized how interactive design can help users engage more deeply with virtual objects, turning passive viewing into active exploration. The project allowed me to combine technical learning with cultural storytelling.

Deployment & Publication

The project has been fully implemented and is ready for deployment. All pages, 3D models, and interactive features are well-integrated and tested. The site is structured to run as a static web application, making it suitable for hosting on platforms such as GitHub Pages or Netlify. This project will be included in my personal portfolio to showcase both technical and creative skills. It can also serve as a demo for interactive cultural experiences or 3D product showcases. Future plans include refining the UI/UX for mobile devices, expanding the collection of models, and integrating real-time lighting or animation libraries for more immersive interactions.
